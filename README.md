# Lung_Cancer-prediction using Machine Learning

Lung Cancer is one of the leading cause of cancer-related deaths wordlwide.Early detection can significantly improve survival rates.
This project focuses on predicting lung cancer using machine learning algorithms based on patient medical and liferstyle data.

The model analyzes various risk factors such as smoking habits, age, chronic diseases and symptoms to classify whether a person is likely to have lung cancer.

#Technologies Used
Programming language: Python
Libraries & Frameworks:
. Pandas
. Scikit-learn
. Random_Forest classifier
. Gradient_Boosting classifier
Development Environment : Google Colab

#Evaluation 
Accuracy:
.Random_Forest classifier : 91.3%
.Gradient_Boosting classifier: 90.1%
